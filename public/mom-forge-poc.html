Scope

What This Tool Covers

MoM Forge is scoped as an internal productivity tool for teams that need consistent, fast, and auditable meeting documentation.

🎯 Primary User

Meeting facilitators, project managers, business analysts, and executive assistants who document meetings regularly.

📄 Input

Any meeting transcript in .txt, .pdf, or .docx format. Manual or auto-generated transcripts both supported.

📊 Output

Professionally formatted Word (.docx) document or Excel (.xlsx) spreadsheet with structured sections. PDF export also available.

🏢 Scale

Designed for multi-user, multi-team access. Each user has their own account, job history, and template library.

🔒 Security

Row-level security via Supabase. Each user can only access their own files, jobs, and outputs. Admin panel for platform oversight.

☁️ Deployment

Fully cloud-hosted on Vercel (serverless). No infrastructure management required. Accessible from any browser.

Optional Template

Upload your organisation's existing MoM template (.xlsx, .docx, .pdf) and the output will match your exact structure. No template? A professionally formatted Word document is auto-generated.

🧠

Structured Extraction

The AI returns validated JSON covering: meeting title, date, time, facilitator, attendees (with org), agenda, discussion topics, decisions, action items (with owner & due date), and next meeting details.

🔄

Real-time Job Tracking

Each generation is a tracked "job" with a live event log. Users can see each pipeline stage (uploaded → processing → completed), retry failed jobs, and access full history.

Capabilities

What This Tool Does & Does Not Do

Clear boundaries of the current POC scope.

✅ What It Does

✅

Extract attendees, decisions, action items (owner + due date), discussion topics from any transcript

✅

Generate a professionally formatted Word (.docx) document without any template

✅

Generate a structured Excel (.xlsx) file matched to your organisation's template format

✅

Export PDF from any completed job

✅

Store and manage up to 5 reusable templates per user with a default selection

✅

Maintain full job history with downloadable outputs and event log per job

✅

Support multi-user access — any email domain, no domain restrictions

✅

Provide admin panel with user management, health monitoring, and platform stats

✅

Append AI disclaimer to all generated documents for responsible use

✅

Works entirely in the browser — no software installation required

❌ What It Does Not Do

❌

Record or transcribe meetings in real-time — a transcript must be provided

❌

Process audio or video files (no speech-to-text in current scope)

❌

Send MoM automatically via email to attendees

❌

Integrate with calendar systems (Outlook, Google Calendar) for scheduling

❌

Support electronic signatures or approval workflows

❌

Process transcripts in languages other than English (current scope)

❌

Connect to Zoom, Teams, or Meet directly for auto-transcript fetch

❌

Guarantee 100% accuracy — AI extraction must be reviewed by the user before sharing

⚠

AI Accuracy Advisory: All MoM outputs are generated by artificial intelligence based on the provided transcript. Users must review and verify all information — including names, dates, decisions, and action items — for accuracy and completeness before sharing on any platform or using for official purposes.
